# Слухай

Українська PWA-читалка для EPUB, PDF і TXT з системним озвученням та опційним перекладом через LibreTranslate.

## Запуск

З каталогу проєкту запустіть будь-який статичний HTTP-сервер, наприклад:

```sh
python3 -m http.server 8080
```

Потім відкрийте `http://localhost:8080`. Для встановлення на телефоні опублікуйте каталог через HTTPS і виберіть «Додати на початковий екран» у браузері.

PDF та EPUB модулі вперше завантажуються з CDN і після цього кешуються PWA. TXT працює одразу. Якість і доступність українського голосу залежить від операційної системи.

## Мобільні проєкти

У репозиторії вже є готові оболонки `android/` та `ios/` на Capacitor із нативним Text-to-Speech. Вебресурси синхронізовані в обидві платформи.

```sh
npm install
npm run native:sync
npm run android
npm run ios
```

Для Android потрібна Android Studio з Android SDK. Для iOS потрібні повний Xcode та CocoaPods; збірка iOS виконується лише на macOS.
